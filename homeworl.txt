1# What is an IDE? (Что такое IDE?)

IDE (integrated development environment) интегрированная среда разработки - программа для написания, тестирования и запуска кода.
В IDE есть редактор кода, утилиты для запуска и тестирования кода, также расширения для удобства
Например Git Graph для перемещения по ветки git



Write three reasons why professional programmers prefer to use an IDE like VSCode (Напишите три причины, по которым профессиональные программисты предпочитают использовать IDE вроде VSCode)

Причины использовать IDE:
2. Полная кастомизация под себя, то есть ты можешь поставить любую цветовую рацветку приятную для твоего глаза, что влияет на качество работы
1. Скорость работы, ты можешь делать вещи которые требуют много времени быстро, использую горячие клавиши. Например: запуск кода, разметка HTML, создание тегов
3. Ты можешь работает над несколькими проектами при этом перемещеясь между ними. Например работал в одном проекте открыл пару файлов и работал над ними, перешел в другой проект и сделал тоже самое, ты можешь вернуться обратно и те файлы снова будут открыты
